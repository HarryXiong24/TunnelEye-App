# Tunnel_Eye_App

## Description

Tunnel Eye App, typescript version.

## Project setup

``` bash
# Install dependencies
yarn install
```

``` bash
# Compiles and hot-reloads for development
yarn serve
```

``` bash
# Compiles and minifies for production
yarn build
```

``` bash
# Lints and fixes files
yarn lint
```
